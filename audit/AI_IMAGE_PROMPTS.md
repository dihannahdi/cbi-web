## Technical Specifications

### Recommended Sizes:
- Featured Image (Hero): 1200 x 630px (OG Image standard)
- Blog Content: 800 x 450px
- Infographics: 1200 x 1200px (Instagram) or 1080 x 1920px (Story)
- Product Images: 1000 x 1000px (square)

### File Formats:
- Primary: WebP (for web optimization)
- Fallback: JPEG (85% quality)
- PNG for images with transparency

### Alt Text Guidelines:
All images should have descriptive alt text including:
- Target keywords naturally incorporated
- Description of what's shown in the image
- Company name where appropriate

Example:
```
Alt: "Pupuk hayati cair Flora One sedang diaplikasikan pada tanaman padi di sawah Indonesia - PT Centra Biotech Indonesia"
```
